length : 객체(문자열)의 길이

replace(a,b) : a를 b로 치환

replaceAll(a,b) : a를 b로 모두치환

toUpperCase() : 대문자로 치환

toLowerCase() : 소문자로 치환

concat(a, b) : a와 b를 결합한다.

trim() : 양쪽 공백을 삭제한다.

trimStart() : 앞쪽 공백만을 삭제한다.

trimEnd() : 뒷쪽 공백만을 삭제한다.

split() : 구분자를 기준으로 잘라준다. 잘라진 각각의 요소는 배열로 저장된다.

indexOf() : 특정문자의 위치값을 찾아준다.(찾는 값이 없으면 -1 출력)

lastIndexOf() : 특정문자의 위치를 뒤에서부터 찾아서 위치값을 찾아준다.

search() : 특정문자의 위치값을 찾아준다(찾는 값이 없으면 -1 출력)

match() : 찾는 문자열이 있으면 '문자열'을 반환, 없으면 '공백'을 반환한다.

includes() : 찾는 문자열을 포함하고 있으면 'true'를 반환, 없으면 'false'를 반환한다.
